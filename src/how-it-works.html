How does BlockVote works?

by Student of Chandigarh University

Posted on April 23, 2022

Standing in the queue and waiting for your turn to cast vote is a part of a history now.

In every democratic country elections occur and it is taken either by using ballet paper or e-voting which is centralised. E-voting is in use for a long period of time because it has some benefits over Paper based systems but this conventional e-voting system is centralized. A particular central administration manages the entire system and has control over the database and due to this has the possibility of manipulating data. But by using blockchain technology the e-voting system which I have created is decentralised. Blockchain is a disruptive technology of this era which has potential to solve all the problems of the e-voting system.

Blockchain based voting is the upcoming way of casting vote where all the votes will be stored in the Blockchain network, which provide security, voter authentication, accessibility and voter anonymity .

This is still under various trials! so please ignore bugs.

How to use?

1. Install MetaMask Browser Extension.

2. Make sure that the Ethereum Account which you've been assigned is imported into MetaMask.

3. Visit the Cast Vote Page and cast your vote
